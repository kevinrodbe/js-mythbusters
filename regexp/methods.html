    • Use the correct RegExp method

    First, always save the RegExp expression in a variable as:

    var reContains = /(?:^| )foo(?: |$)/

    Second, in general terms:

    • Use .test if you want a faster boolean check.
    • Use .match to retrieve all matches when using the g global flag.

    It’s possible that you can gain an extra of perfomance using String method to match RegExp instead of RegExp to match String. It’s depends of your case of use and of your JavaScript Engine. Check test#1, test#2 & test#3 benchmarks.
